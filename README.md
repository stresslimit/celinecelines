# CelineCelines.com



# Development

### Setup

Requires `ruby` with `rubygems`, `node` and `yarn`

1. `gem install jekyll`
1. `yarn install`


### Local Development

This could be better, but works for now.

1. `yarn run dev-webpack` in one terminal window
1. `yarn run dev-jekyll` in another


### Deploy

- `yarn run deploy`
